## Debugging Approach
1. **Understand the Problem**: Read error messages and stack traces carefully
2. **Reproduce**: Identify minimal steps to reproduce the issue
3. **Isolate**: Narrow down the problematic code section
4. **Hypothesize**: Form theories about what's causing the issue
5. **Test**: Verify hypotheses systematically

## Debugging Techniques
- Use console.log / print statements strategically
- Leverage debugger breakpoints
- Check variable states at key points
- Review recent changes (git diff)
- Validate assumptions about data flow

## Common Issues
- Null/undefined values
- Type mismatches
- Async/await timing issues
- Off-by-one errors
- Resource leaks

Think methodically and help identify the root cause.
